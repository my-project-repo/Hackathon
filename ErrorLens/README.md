🖥️ Installation Guide
<br/>
Step 1 — Install Visual Studio Code
<br/>

If you do not already have VS Code installed:

<br/>

👉 Download VS Code from:

Visual Studio Code Official Website

<br/>

Install it normally for your operating system.

<br/>
<br/>
📦 Installing ErrorLens (.vsix)
<br/>

You already have the extension file:

<br/>
errorlens-0.0.1.vsix
<br/>
<br/>
Method 1 — Install Using VS Code UI (Recommended)
<br/>
Open VS Code
<br/>

Then:

<br/>
Open the Extensions Sidebar
Click the ⋯ (three dots) menu in the top-right
Click:
<br/>
Install from VSIX...
<br/>
Select:
<br/>
errorlens-0.0.1.vsix
<br/>
Reload VS Code if prompted
<br/>

✅ ErrorLens is now installed

<br/>
<br/>
Method 2 — Install Using Terminal
<br/>

Open terminal inside the folder containing the .vsix file.

<br/>

Run:

<br/>
code --install-extension errorlens-0.0.1.vsix
<br/>
<br/>
If code Command Does Not Work
<br/>

Inside VS Code:

<br/>
Press:
<br/>
Mac
Cmd + Shift + P
<br/>
Windows/Linux
Ctrl + Shift + P
<br/>
Search:
<br/>
Shell Command: Install 'code' command in PATH
<br/>
Press Enter
<br/>

Then retry:

<br/>
code --install-extension errorlens-0.0.1.vsix
<br/>
