# 🧠 What LogicLens Detects

LogicLens focuses on logic mistakes that traditional compilers and linters often fail to explain clearly.

<br/><br/>

## 🔴 Common Programming Errors

- Off-by-one errors  
- Infinite loops  
- Incorrect loop updates  
- Empty loop bodies  
- Unreachable code  
- Assignment inside conditions  
- Missing return statements  
- Duplicate conditions  
- Floating point equality mistakes  
- Null or undefined access risks  
- Stack overflow risks  

<br/><br/>

## 🟡 Data Structure & Algorithm Mistakes

- Binary search midpoint overflow  
- Sliding window not shrinking  
- Dynamic programming state overwrite  
- Missing hashmap existence checks  
- Array mutation during iteration  
- Missing recursion base cases  
- Missing edge case handling  
- Modulo by zero risks  

<br/><br/>

## 🔵 Educational Guidance

For every issue detected, LogicLens provides:

- Clear explanations  
- Suggested fixes  
- Learning hints  
- Related DSA concepts  
- Topic mapping for interview preparation  
- Beginner-friendly mentoring

# 🖥️ Installation Guide

<br/>

## Step 1 — Install Visual Studio Code

If you do not already have VS Code installed:

👉 Download VS Code from:  
https://code.visualstudio.com/

Install it normally for your operating system.

---

## 📦 Installing LogicLens (.vsix)

You already have the extension file:
logiclens-0.0.1.vsix

---

## Method 1 — Install Using VS Code UI (Recommended)

- Open VS Code  
- Go to Extensions Sidebar  
- Click the ⋯ (three dots) menu in the top-right  
- Click **Install from VSIX...**  
- Select `logiclens-0.0.1.vsix`  
- Reload VS Code if prompted  

✅ LogicLens is now installed

---

## Method 2 — Install Using Terminal

Open terminal inside the folder containing the `.vsix` file:

```bash
code --install-extension logiclens-0.0.1.vsix
