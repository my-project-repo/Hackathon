# 🧠 What LogicLens Detects

LogicLens focuses on logic mistakes that compilers and normal linters cannot explain.

---

## 🔴 Common Programming Errors

- Off-by-one errors  
- Infinite loops  
- Incorrect loop updates  
- Empty loop bodies  
- Unreachable code  
- Assignment inside conditions  
- Missing return statements  
- Duplicate conditions  
- Floating point equality mistakes  
- Null or undefined access risks  
- Stack overflow risks  

---

## 🟡 Data Structure & Algorithm Mistakes

- Binary search midpoint overflow  
- Sliding window not shrinking  
- Dynamic programming state overwrite  
- Missing hashmap existence checks  
- Array mutation during iteration  
- Missing recursion base cases  
- Missing edge case handling  
- Modulo by zero risks  

---

# 🖥️ Installation Guide

## Step 1 — Install Google Chrome

Download Chrome if not already installed:

👉 https://www.google.com/chrome/

---

## Step 2 — Install LogicLens Extension

### Method 1 — Load Unpacked Extension (Developer Mode)

1. Open Chrome
2. Go to:
chrome://extensions/
3. Enable **Developer Mode** (top right)
4. Click **Load unpacked**
5. Select the LogicLens extension folder

---

### Method 2 — Install via ZIP (if packaged)

1. Download LogicLens extension ZIP
2. Extract it
3. Load extracted folder using **Load unpacked**

---

## Step 3 — Start Using

- Open any supported coding platform
- Start solving problems
- LogicLens will automatically detect code
- Errors appear in real time in floating panel
